# java-cli-gradle-pmd-jacoco-jbehave-testng-hello-world

## Description
A POC app using testng,
jacoco, pmd, jbehave framework.

## Tech stack
- java
- gradle
  - testng
  - jbehave
  - jacoco
  - pmd

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
